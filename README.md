# 🏡 ImmoNest – Real Estate Platform

A modern full-stack real estate web application built with **Laravel, Vue.js, and Inertia.js**, designed to demonstrate scalable architecture, clean code practices, and production-ready UI development.

---

## 🚀 Overview

**ImmoNest** is a portfolio project that simulates a real estate management platform. It allows users to manage property listings with advanced filtering, sorting, and full CRUD functionality.

The project focuses on building a **real-world SaaS-like experience** using a modern Laravel + Vue stack.

---

## ✨ Features

* 🏠 Property Listings Management (CRUD)
* 🔍 Advanced Filtering (price, date, status)
* ↕️ Dynamic Sorting (ascending / descending)
* 🗑️ Soft Delete & Restore Support
* ⚡ SPA-like experience using Inertia.js
* 🎨 Responsive UI with Tailwind CSS
* 📦 Clean and reusable Vue components

## ⚙️ Installation

```bash
# Clone repository
git clone https://github.com/your-username/immonest-laravel-vue-real-estate.git

cd immonest-laravel-vue-real-estate

# Install PHP dependencies
composer install

# Install JS dependencies
npm install

# Environment setup
cp .env.example .env
php artisan key:generate

# Run migrations
php artisan migrate

# Start development server
php artisan serve
npm run dev
```
